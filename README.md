# One-Liner-Reverse-Shell-Generator-Script
One Liner Reverse Shell Generator Script

Generate One Liner Reverse Shell Commands 

1.bash      2. sh
3.perl      4.py
5.php       6.ruby
7.nc        8.nct (Tradtional without -e)
9.java

Run the bash script. Usage and Shell Command Name will be displayed
